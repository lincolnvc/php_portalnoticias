== Installation ==

### First download the WordPress Polls plugin ZIP file.  
Installing the **WordPress Polls plugin**  

1. Log in to your website administration panel.   
2. Then go to Plugins page Add New > Upload Plugin .    
3. Click "Choose file" then select the - [WordPress Poll plugin](https://wordpress.org/plugins/polls-widget/) zip file. 
4. Then Install and activate the plugin.     
5. Then, from the left panel of Dashboard, click on the **Polls** icon to add and customize Polls
	  
That's all, now you can build Polls and Survey forms. If any problem occurs, contact us at [support@wpdevart.com](mailto:support@wpdevart.com).

== Frequently Asked Questions ==

Our WordPress Poll plugin is user friendly, but here you can see some frequently asked questions that will help you build polls.

= How can I build a poll? =

Go to your administration panel, click on the "Polls" from your admin panel.
Then click on "Polls Manager", click on "Add New" button and then add poll name.
Type poll question and create answers for your poll.
 
= How can I create poll themes? =

Go to your administration panel, click on the "Polls" from your admin panel.
Then click on "Polls Themes", click on "Add New" button and then add poll theme name.
After that set you poll theme and save it.

= Can I add the same poll in different pages? =

Yes. You can even add the same poll on different widgets and posts at the same time.

= How can I add poll in pages/posts? =

First of all you need to create poll and poll theme. Then you must go to your post/page and add poll clicking on poll shortcode from your post/page editor.
After that you'll see a window, choose your poll and then choose theme for your poll.

= How can I add poll on widget? =

Go to your widgets page, then add our [WordPress Poll plugin](https://wordpress.org/plugins/polls-widget/) widget to your sidebar.
After that choose your poll and then choose theme for your poll.

= Can I create poll with images? =

Yes, you can. For adding images you can use WordPress posts and pages editor. It's very simple.

= Can I create poll with movies? =

Yes, that's not a problem too.

== Other Notes ==

### Creating WordPress Polls

*   Poll name - Type here Poll name.
*   Question - Type here Poll question. You can use images and edit your question text using WordPress standard posts editor.
*   Choose the answer type - Click on Multiple answers to choose Poll with Multiple choice answers or click on single answers to choose Poll with Single choice answers.
*   Answers - Type here Poll answers. Type the answer, then click anywhere and the new answer field will appear. Click remove to remove any answer you need. 

Then just save the Poll that you created.

The second option page is "Polls design". On this page you can create design for your WordPress Polls.

### [WordPress Polls](https://wordpress.org/plugins/polls-widget/) design

*   Poll theme name - Type here the Poll theme name.

There are two global options, simple design options and hover design options. When you change the simple design option the hover design option will
change automatically, but when you change the hover design option the simple design option doesn't change. We add this functionality for 
users convenience.

*   Poll height - Select the Poll height. Then select the Poll height when hovering.   
*   Poll background color - Select the Poll background color. Then select the Poll background color when hovering.
*   Poll background opacity - Select the Poll background opacity. Then select the Poll background opacity when hovering.
*   Distance among Poll answers - Select distance among Poll answers.
*   Answer font size - Select the Poll answer font size. Then select the Poll answer font size when hovering.   
*   Answer distance from top - Select the Poll answer distance from top. Then select the Poll answer distance from top when hovering.   
*   Answer distance from left - Select the Poll answer distance from left. Then select the Poll answer distance from left when hovering.   
*   Background color corresponding counts - Select the Poll background color corresponding counts. Then select the Poll background color corresponding counts when hovering.
*   Answer color - Select the Poll answer color.  Then select the Poll answer color when hovering.

*   Answer font style - Select the Poll answer font style. Then select the Poll answer font style when hovering.
*   Poll border width - Select the Poll border width. Then select the Poll border width when hovering.

*   Count distance from left - Select the Poll count distance from left. Then select the Poll count distance from left when hovering.
